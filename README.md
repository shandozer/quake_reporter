# quake_datafeed.py
Gathers global quake (and tsunami) information and presents to user.

## Version 0.1.0
Prints unsorted results to console for magnitudes **m** and above

## Optional Flags
`-m <magnitude: 1.0, 2.5, or greater>`
_(default: 2.5+)_

`-t <time frame: day, hour, week, or month>` 
_(default: 'week')_
